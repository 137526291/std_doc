前期最常用命令：

初始化本地仓库

::

    git init


::

    git add .   //添加目录下所有东西到暂存区


::

    git commit -m "提交记录，新增xxx 修复xxx 。。。等"

::

    git push origin master   //推算到远程仓库


::

    git pull origin master   //拉取远程仓库到本地
